# Laboratory Log: SSP Case Study 1 — Session 3

## 1. Objective
Design Capon spatial filters for each of the 5 detected sources, estimate QPSK symbols and decode the hidden ASCII messages.

## 2. Activity
Computed Capon beamformer weights as w_i = Rinv * a(theta_i) / (a'(theta_i) * Rinv * a(theta_i)) for each of the 5 DoAs. Applied filters to received signal: S_hat = W' * Y. Normalized power per user. Extracted bits from real and imaginary parts using sign function. Decoded bits to ASCII using reshape and bin2dec.

## 3. Observations
Users 1 and 5 show clean QPSK constellations — well separated clusters. Users 2, 3, 4 show distorted constellations due to closely spaced DoAs and lower SNR. Decoded messages confirm quality differences: Users 1 and 5 produce readable sentences, Users 2-4 show decoding errors. Full messages recovered for all 5 users.

## 4. Decisions
Applied power normalization per user before constellation plot for consistent scaling. Used sign() function for bit extraction as specified in the statement.
