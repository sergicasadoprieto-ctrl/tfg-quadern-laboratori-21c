# Diagonal Loading: Why It Matters

## Core Idea
When inverting the sample covariance matrix R_hat, numerical instability can occur if the matrix is ill-conditioned. Diagonal loading adds a small regularization term to ensure stable inversion: R_reg = R_hat + alpha * I, where alpha = 1e-6 * trace(R_hat) / M.

## Why It Matters for This Project
Without diagonal loading, the Capon filter weights become unreliable and the spatial spectrum shows artifacts. This is especially critical near endfire directions where the covariance matrix is most sensitive to noise.
