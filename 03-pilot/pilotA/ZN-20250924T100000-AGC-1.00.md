# Algebraic Note: Covariance Matrix and Diagonal Loading

## Core Idea
Y (MxN) matrix contains different observations of the
system of sensors. Sample covariance matrix:
R_hat = (1/N) * Y * Y^H

Is (Y * Y^H) invertible? Only if N >= M (columns > rows).
When R_hat is not full rank, diagonal loading is applied:
(R_hat + alpha*I)^-1 where alpha > 0
