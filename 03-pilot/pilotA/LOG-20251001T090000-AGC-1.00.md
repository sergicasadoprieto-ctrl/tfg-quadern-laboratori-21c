# Laboratory Log: SSP Case Study 1 — Session 1

## 1. Objective
Implement the sample covariance matrix estimation for a ULA receiver with M=10 antennas and N=392 snapshots.

## 2. Activity
Loaded receivedSignals.mat dataset. Implemented sample covariance matrix as R_hat = (1/N) * Y * Y'. Generated steering matrix A for angles between -90 and 90 degrees with 0.01 degree resolution (18001 angles total).

## 3. Observations
Vectorized implementation works correctly. Steering matrix A has dimensions 10x18001 as expected. Applied diagonal loading with alpha = 1e-6 * trace(R_hat)/M to ensure stable matrix inversion even though M < N in this case.

## 4. Decisions
Decided to apply diagonal loading as good practice regardless of matrix conditioning. Kept alpha proportional to average power as recommended in theory notes.
