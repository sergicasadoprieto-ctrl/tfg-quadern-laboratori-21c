# Laboratory Log: Survey Data Processing

## 1. Objective
Process raw survey data from Google Forms export and generate visualisations for Power BI dashboard and GitHub Pages.

## 2. Activity
Raw Excel file exported from Google Forms processed using Python (Google Colab). Multiple-choice responses cleaned and expanded. Three CSV files generated: responses_clean.csv, multi_select_expanded.csv, summary_counts.csv. Power BI dashboard created with QL21 theme applied. Learnt in job.

## 3. Observations
Main processing challenge: multiple-choice responses stored as concatenated strings in single cells. Substring matching used instead of delimiter splitting. Power BI dashboard shows six visualisations covering all survey sections.

## 4. Decisions
CSV chosen as output format for tool-agnostic compatibility. Power BI used for interactive dashboard. HTML page to be published via GitHub Pages.

## 5. Next Steps
- Publish HTML results page to GitHub Pages
- Send toolkit to survey participants who left contact details
